清明前夕小记

发表于 2017-03-31   |   分类于 杂谈

说来也惭愧，去年下半年入职到现在，技术层面的归档做的很差。一是忙于新公司的业务，二是整个人在生活上也懒散了不少，说这些也不是为了给自己开脱。来了上海之后，确实心态发生了变化，加上生活中发生的琐碎事件，浮躁了很多。特此在清明前夕，标列一下17年的计划，总结一下这几个月来的得失。

2017年的计划

  1. 8月底之前瘦到 72kg，17年能跑一次马拉松
  2. 一年阅读 50 本书，挑一些喜欢的书写几篇读后感
  3. 练好行楷
  4. 写两到三个第三方库, GitHub 一年 commit 150+
  5. 起码拿一个季度考评 A
  6. 上架 一两个 App
  7. 出去走走，在下雨天去一两个古镇，体验一下烟雨朦胧的胜景

制定计划的目的

  • 关于体重，这个是老梗，从小到达就没瘦过，本身自己也是从事it一行的，很多前辈到了一定年纪，身体或多或少有点问题。而自己在有时候也有点力不从心的感觉，所以旧事重提，减脂放在第一位了
  • 换了新的工作环境，周围人来人往，或多或少有交际，谈吐方面愈发觉得自己言辞匮乏，而且心态变化很大，确实需要一定的阅读量来提升自己
  • 写的一手草书，好看就算了，关键还很丑，而且也确实因为笔迹丑陋错失了很多机会。周围亲近的人不少也提醒过我，所以这个计划也提上来了~
  • 开源几个第三方库，本来是去年就打算做的，但是为了来上海工作这事也耽搁了。目前已经有两个想法，一个是基于蓝牙模块的库，对应公司的业务，二是 tab 标题轮动切换的库
  • 公司年底的时候，薪酬制度做了调整了，拿到A的话，是对工作质量上的一定肯定。其实讲道理，为了养家糊口，也确实应该树立这个目标…

说说上半年做了哪些事?

1.体重下来了

二月份到前一阵子控制饮食加上些许运动，体重一度降到了80.6kg，感慨下不是瘦不下来，而是自己在饮食太放纵，在生活上太放肆了

2.阅读了几本书

  • 孤独深处
  • 长安十二时辰
  • 最好的告别 (在读) 这本书确实不错，爱不释手
  • 我喜欢这个功利的世界

在买书的时候，根据自己的需要买了些书，这几年家里发生的事比较大，心态问题挺严重的，也想不到合适的时候去倾诉。这几本书读的很快，不过目前在读的 <最好的告别> 这本，爱不释手，实在舍不得读快了，上半年抽空写几篇读书小记吧

未来几个月的憧憬

人在窘迫的时候总会缅怀过去，总是希望能回到过去多好，重走一遍人生，我也一样，免不了俗(于我而言，最好的时光应该是初中了吧)

今年一年的计划定的挺好看的，但是17年已经过去了四分之一了。今天是
三月底的最后一天，想去做就去做，实现自己的计划吧!
