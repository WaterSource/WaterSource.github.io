iOS 11 更新了哪些细节?

发表于 2017-09-24   |   分类于 技术

导航栏的变更

Navigation Bars

一个导航栏出现在一个应用程序界面的顶部，在状态栏下面，并且可以通过一系列的分层屏幕进行导航。当显示一个新的屏幕时，一个后退按钮，通常被标记为前一个屏幕的标题，出现在栏的左边。有时，导航栏的右侧包含控件，比如编辑或Done按钮，用于管理活动视图中的内容。在分屏视图中，一个导航栏可能出现在分屏视图的一个面板中。导航条是半透明的，可能有背景色，当键盘在屏幕上时，可以配置为隐藏，一个手势或视图调整。

鉴定用户

Authentication

要求用户只进行身份验证，比如对体验进行个性化、访问附加功能、购买内容或同步数据。如果你的应用需要认证，那么登录过程要快速、简单、不唐突，这样就不会影响你的应用程序的乐趣。

Face ID and Touch ID

只要有可能，就支持生物测定认证。人脸ID和Touch ID是安全的、熟悉的身份验证方法。如果用户启用了生物识别身份验证，您可以假定他们理解它是如何工作的，感谢它的便利，并且更愿意在可能的情况下使用它。请记住，人们可能会选择在他们的设备上禁用生物认证，因此你的应用应该准备好应对这种情况。

增强现实

Augmented Reality

应用可以使用苹果的增强现实(AR)技术，ARKit，提供沉浸式的、引人入胜的体验，无缝地将现实的虚拟物体与现实世界融合在一起。在AR应用中，该设备的摄像头被用来呈现现实世界的实时画面。三维虚拟物体被叠加在这个视图上，创造出它们实际上存在的错觉。用户可以重新定位设备，从不同角度探索物体，如果合适的话，可以使用手势和动作与他们进行互动。

拖拽手势

Drag and Drop

用一根手指，用户可以移动或复制选定的照片、文本或其他内容，将内容从一个位置拖拽到另一个位置，然后举起手指来删除。

安全区域布局指南

Safe area layout guides

布局指南定义了矩形区域，这些区域在屏幕上实际上并没有明显的显示，但对内容的定位、对齐和间距提供了帮助。该系统包含预定义的布局指南，可以方便地在内容周围应用标准的边缘，并限制文本的宽度以达到最佳的可读性。您还可以定义定制的布局指南。
